# Twitter-Tools
Work in progress - Tools to automate Twitter functionality

![](http://i.imgur.com/JpASdqv.gif)

# How to run
A config.json file in the root directory with your API keys is needed.
```sh
{
    "APIKey": "#",
    "APISecret": "#",
    "oauthToken": {
        "oauth_token": "#",
        "oauth_token_secret": "#"
    }
}
```
Run the following in terminal
```sh
$ npm install
$ npm start
```
